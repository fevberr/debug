✨ Features

Inspection Tabs

Explore different parts of any webpage:

  • Window – View global properties and objects
  • DOM – Inspect elements and structure
  • Links & Forms – Extract interactive elements
  • Stylesheets – View loaded CSS
  • Custom Variables – Track your own data

Actions

Quick tools to interact with data:

  • Copy – Copy output instantly
  • Save – Export results
  • Clear – Reset output
  • Wrap – Toggle text wrapping
  • Refresh – Reload current data

UI & Design

  • Clean interface with consistent SVG icons
  • Tab-based layout for easy navigation
  • Lightweight and responsive

Compatibility

  • Works on any website
  • Fully self-contained (no dependencies)
  • Runs directly from a bookmarklet
